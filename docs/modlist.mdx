# 模组列表

本整合包包含以下模组，全部来自 Modrinth，完美兼容 Forge 1.19.2。

| 类别 | 模组名称 | 功能简介 |
| :--- | :--- | :--- |
| **优化** | 铷 (Rubidium) | 核心渲染优化，大幅提升 FPS |
| | 星光 (Starlight) | 重写光照引擎，根除卡顿 |
| | Oculus | 光影支持，与铷无缝衔接 |
| | 实体剔除 (Entity Culling) | 不渲染视角外实体，提升性能 |
| | FerriteCore | 内存优化，减少卡顿 |
| | 钌 (RuOK) | 动态实体渲染优化 |
| **辅助** | JEI (Just Enough Items) | 查看所有物品配方与用途 |
| | Jade | 显示看向的方块/实体信息 |
| | 苹果皮 (AppleSkin) | 显示食物饱食度恢复量 |
| | Xaero的小地图 | 轻量小地图，支持路径点 |
| | 鼠标手势 (Mouse Tweaks) | 鼠标拖拽快速转移物品 |
| | JEI拼音搜索 | 支持拼音搜索物品 |
| **世界** | Terralith | 重塑主世界地形，无需新世界 |
| | Tectonic | 生成更真实的山脉与峡谷 |
| | YUNG's 系列 (Better Dungeons, etc.) | 改良地牢、要塞等结构 |
| | 你将去的生物群系 (Oh The Biomes You'll Go) | 添加众多新群系与建材 |
| **战斗** | 大规模毁灭性Boss (Bosses of Mass Destruction) | 增加数场史诗级Boss战 |
| | mutant怪物 (Mutant Monsters) | 增加原版怪物的 mutant变种 |
| | 末影龙战重制 (Ender Dragon Fight Remastered) | 让末影龙成为真正的最终Boss |
| | 魂类宇宙 (Souls like Universe) | 加入类魂武器与Boss |
| | 史诗战斗 (Epic Fight) | 为战斗添加华丽动画 |
| **科技** | 机械动力 (Create) | 趣味自动化，机械美学核心 |
| | 应用能源2 (Applied Energistics 2) | 物流与存储终极方案 |
| | Hex 施法 (Hex Casting) | 基于十六进制的复杂魔法 |
| **生活** | 农夫乐事 (Farmer's Delight) | 扩展烹饪与农业 |
| | Quark | 添加众多原版风味小特性 |

> 💡 所有模组均可在 Modrinth 上搜索项目ID（见模组名称后的括号）找到对应版本。
