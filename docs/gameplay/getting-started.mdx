# 新手入门指南

欢迎来到 **Together Epoch**！这里有一些快速上手的建议，帮助你更快融入这个世界。

## 第一步：熟悉基础模组

进入游戏后，建议先了解几个核心辅助模组：
- **JEI (Just Enough Items)**：按 `R` 键查看物品合成配方，按 `U` 键查看物品用途。这是生存的“说明书”。
- **Jade**：准心对准方块或生物时，屏幕上方会显示其名称和额外信息（如箱子内容、生物血量）。
- **Xaero的小地图**：按 `J` 键可以设置路径点，方便标记重要地点。

## 第二步：初期发展路线

1. **收集资源**：与原版一样，先撸树、挖石头。注意新群系带来的新木材种类（如红木、杨木），它们各有特色。
2. **寻找村庄**：使用小地图寻找村庄，获取初期物资和床。**YUNG's Better 系列** 让村庄地牢更有探索价值。
3. **制作工具**：利用 JEI 查看新材料的用途。**Quark** 添加了各种实用小工具（如锁链、木桶变种），可以尝试制作。

## 第三步：小心夜晚

新怪物和变种怪物会让你措手不及：
- **Mutant 怪物**：原版僵尸、骷髅等有概率生成 mutant 变种，它们体型更大、技能更强。
- **野外Boss**：在特定地形（如荒地、森林）可能遇到 **Bosses of Mass Destruction** 的小型Boss，初期建议绕行。

<Callout type="warning">
  🛡️ 早期建议与队友结伴探索，单人玩家至少准备好铁套再外出。
</Callout>

## 第四步：选择你的方向

本整合包为不同玩家提供了多条发展路径：
- **战斗派**：可尽早寻找 BOSS 挑战，获取强力装备。
- **科技派**：从 **Create** 入手，搭建简单的机械自动化生产线。
- **魔法派**：研究 **Hex Casting** 的咒术，用复杂魔法解决难题。

无论选择哪条路，别忘了和队友分工合作，效率翻倍！
