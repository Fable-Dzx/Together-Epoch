# BOSS攻略指南

**Together Epoch** 新增了多个史诗级BOSS，以下是部分BOSS的简要攻略。

## 1. 荒原巨像 (来自 BMD)
- **出现地点**：荒地生物群系
- **特点**：巨大的石像，会召唤岩石傀儡，有范围砸地攻击。
- **攻略建议**：
  - 保持移动，避免被砸地击中。
  - 优先清理召唤的傀儡，否则会被围攻。
  - 推荐使用远程武器（弓、弩）在攻击间隙输出。

## 2.  mutant 巨人 (来自 Mutant Monsters)
- **出现地点**：平原、森林，夜晚概率生成
- **特点**：僵尸的 mutant 变种，体型巨大，会投掷泥土块并召唤普通僵尸。
- **攻略建议**：
  - 利用地形卡位，避免被投掷物击中。
  - 准备一把高伤害武器（如钻石剑+附魔），快速削减其血量。
  - 团队配合时，一人吸引仇恨，其他人从背后输出。

## 3. 重制末影龙 (来自 Ender Dragon Fight Remastered)
- **出现地点**：末地
- **特点**：战斗分阶段进行，龙会召唤更多末影水晶，攻击模式更复杂。
- **攻略建议**：
  - 准备足够的慢落药水和末影珍珠，应对龙的新击飞技能。
  - 团队分工明确：专人破坏水晶，专人输出龙，专人治疗。
  - 建议先提升装备（附魔钻石套或下界合金套）再挑战。

## 4. 魂系Boss (来自 Souls like Universe)
- **出现地点**：特定结构（如废弃城堡）或使用召唤物召唤
- **特点**：动作敏捷，攻击带有 Debuff，非常考验操作。
- **攻略建议**：
  - 学习其攻击前摇，熟练掌握翻滚（如果安装了翻滚类模组）或走位躲避。
  - 利用 **Epic Fight** 的连击系统进行反击，打出硬直。
  - 单人挑战难度极高，推荐组队轮流吸引仇恨。

<Callout type="tip">
  🏆 击败BOSS后掉落的特殊材料可用于制作强力装备和饰品，值得一试。
</Callout>
