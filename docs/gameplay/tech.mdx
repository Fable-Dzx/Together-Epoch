# 科技与魔法发展路线

本整合包提供了两条主要的技术路线：**科技自动化** 和 **魔法体系**，你可以专精其一，也可两者兼修。

## ⚙️ 科技路线：机械动力 (Create)

**Create** 是本包的核心科技模组，以应力、旋转、齿轮为核心，实现各种自动化。

### 初期目标
- 制造 **水车** 或 **风车**，获得基础应力。
- 搭建 **粉碎轮** 处理矿石，提高矿物产量。
- 制作 **机械手** 实现简单物品转移。

### 中期发展
- 利用 **动力轴承** 建造移动式农场或采矿机。
- 引入 **火车系统**，连接多个基地，实现远距离物流。
- 研究 **配方自动化**，自动合成复杂物品（如高级合金）。

### 后期进阶
- 结合 **Applied Energistics 2**（AE2），建立大型存储网络，管理海量物品。
- 用 **Create** 的机械配合 AE2 的接口，实现全自动化生产。

## 🔮 魔法路线：Hex Casting (Hex 施法)

**Hex Casting** 是一款硬核魔法模组，通过编写十六进制“咒文”来施法。

### 入门要点
- 制作 **书与笔**，开始学习基础“动作”。
- 在 JEI 中查看各种“模式”和“数字”的拼写方式。
- 从简单的“放置方块”、“破坏方块”开始练习。

### 常用咒术示例
- **远程攻击**：编写投射物咒文。
- **传送**：编写传送咒文，配合坐标实现瞬移。
- **自动化**：编写“傀儡”咒文，让魔法代替机械工作。

<Callout type="info">
  📖 Hex Casting 的学习曲线较陡，建议先在创造模式试验，或参考网上教程（B站有系列视频）。
</Callout>

## 🤝 科技与魔法协作

在多人游戏中，可以让擅长科技的玩家负责资源生产和存储，擅长魔法的玩家负责防御、传送和特殊制造。两者结合往往能产生意想不到的效果——例如用魔法远程控制机械开关，或用 AE2 网络为魔法提供稳定的物品来源。
